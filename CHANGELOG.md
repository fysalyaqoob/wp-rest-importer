# Changelog

All notable changes to WP REST Importer are documented here.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
Versioning follows [Semantic Versioning](https://semver.org/)

## [1.0.0] - 2025-01-01
### Added
- Import posts and pages from any public WordPress site via REST API
- Automatic category and tag creation
- Featured image sideloading to Media Library
- In-content image downloading and URL rewriting
- Year/month folder structure matching WordPress uploads
- Original publish date and post status preserved
- Overwrite existing posts by slug — safe to re-run
- Author assignment with user dropdown
- Reassign Authors tool with postmeta matching
- Live progress bar with AJAX batching (5 posts per batch)
- Import log showing Created / Updated / Failed per item
- Gutenberg and Classic content auto-detection
- Yoast SEO and RankMath meta description import
- Clean minimal admin UI with WordPress Dashicons
- Docs and Support links in admin header
- Footer CTA linking to fysalyaqoob.com
